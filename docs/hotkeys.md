# Хоткеи

Источник: `lua/keys/maps.lua`.

## Obsidian
- `<leader>oo` — `:ObsidianOpen` (Открыть в Obsidian GUI)
- `<leader>on` — `:ObsidianNew` (Новая заметка)
- `<leader>ot` — `:ObsidianTemplate` (Вставить шаблон)
- `<leader>od` — `:ObsidianToday` (Daily: сегодня)
- `<leader>oy` — `:ObsidianYesterday` (Daily: вчера)
- `<leader>om` — `:ObsidianTomorrow` (Daily: завтра)
- `<leader>ob` — `:ObsidianBacklinks` (Backlinks)
- `<leader>ol` — `:ObsidianLinks` (Ссылки из заметки)
- `<leader>os` — `:ObsidianSearch` (Поиск по vault)
- `<leader>oq` — `:ObsidianQuickSwitch` (Quick switch)

## Локальные команды заметок
- `<leader>oc` — `:VaultInbox` (Capture в inbox)
- `<leader>tt` — `:ToggleCheckbox` (Toggle checkbox)
- `<leader>vf` — `:VaultFind` (Vault: find files)
- `<leader>vg` — `:VaultGrep` (Vault: grep text)

## Telescope (общие)
- `<leader>ff` — `:Telescope find_files`
- `<leader>fg` — `:Telescope live_grep`
- `<leader>fb` — `:Telescope buffers`

## Markdown Preview
- `<leader>mp` — `:Glow`
