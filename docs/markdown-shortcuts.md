# Шорткаты для Markdown (LuaSnip)

Источник сниппетов: `lua/snippets/markdown.lua`.

## Как использовать
- Работает в буферах с `filetype=markdown`.
- Введите триггер и нажмите `<Tab>` для разворачивания.
- `<Tab>` / `<S-Tab>` переходят по плейсхолдерам сниппета.

## Триггеры
- `h1` — `# Заголовок`
- `h2` — `## Заголовок`
- `h3` — `### Заголовок`
- `ul` — маркированный список из 3 пунктов
- `ol` — нумерованный список из 3 пунктов
- `todo` — `- [ ] задача`
- `table` — таблица 2x2 с шапкой
- `code` — fenced code block:
  - язык (по умолчанию `bash`)
  - тело кода
- `quote` — `> цитата`
