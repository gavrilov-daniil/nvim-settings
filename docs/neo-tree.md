# Neo-tree

Файловый менеджер для навигации по проекту.

## Как работать
- Открыть/закрыть дерево: `<leader>e`.
- Навигация по файлам и папкам — стандартными клавишами Neo-tree.
- При открытии файла фокус возвращается в редактор.

## Хоткеи
- `<leader>e` — `:Neotree toggle`

## Настройки
Источник: `lua/plugins/neo-tree.lua`
- Окно слева, ширина 32.
- Следовать за текущим файлом: включено.
- Скрытые элементы: отображаются.
