# Neo-tree

Файловый менеджер для навигации по проекту.

## Как работать
- Открыть/закрыть дерево: `<leader>e`.
- Внутри окна Neo-tree используйте стандартные клавиши (см. ниже).
- При открытии файла фокус возвращается в редактор.

## Хоткеи (вне окна)
- `<leader>e` — `:Neotree toggle`

## Клавиши внутри Neo-tree (filesystem source по умолчанию)
### Помощь
- `?` — показать окно помощи со всеми маппингами

### Навигация
- `<` — предыдущий источник
- `>` — следующий источник
- `<bs>` — подняться на уровень выше
- `.` — сделать выбранную папку корнем
- `<space>` — развернуть/свернуть узел
- `<cr>` — открыть файл/папку
- `<2-LeftMouse>` — открыть файл/папку
- `C` — закрыть узел (или родителя)
- `z` — закрыть все узлы
- `[g` — перейти к предыдущему git-измененному файлу
- `]g` — перейти к следующему git-измененному файлу

### Открытие в новых окнах
- `S` — открыть в горизонтальном сплите
- `s` — открыть в вертикальном сплите
- `t` — открыть в новой вкладке
- `w` — открыть с выбором окна (window-picker)

### Preview mode
- `P` — включить/выключить превью
- `l` — сфокусировать окно превью
- `<C-f>` — скролл превью вниз
- `<C-b>` — скролл превью вверх
- `<esc>` — выйти из превью-режима

### Файловые операции
- `a` — создать файл или папку (папка через `/` в конце)
- `A` — создать папку
- `d` — удалить файл/папку
- `i` — показать детали файла
- `r` — переименовать
- `b` — переименовать без расширения
- `y` — копировать в буфер (mark)
- `x` — вырезать в буфер (mark)
- `p` — вставить из буфера
- `<C-r>` — очистить буфер
- `c` — копировать (диалог)
- `m` — переместить (диалог)

### Вид и сортировка
- `H` — показать/скрыть скрытые файлы
- `R` — обновить дерево
- `o` — меню сортировки
- `oc` — сортировать по дате создания
- `od` — сортировать по диагностике
- `og` — сортировать по git-статусу
- `om` — сортировать по дате изменения
- `on` — сортировать по имени
- `os` — сортировать по размеру
- `ot` — сортировать по типу

### Фильтрация
- `/` — фильтр (fuzzy)
- `D` — фильтр только по папкам
- `#` — сортировка (fuzzy)
- `f` — фильтр по enter
- `<C-x>` — очистить фильтр

## Дополнительные источники
### Buffers
- `bd` — удалить буфер (`:bdelete`)

### Git status
- `A` — git_add_all
- `ga` — git_add_file
- `gu` — git_unstage_file
- `gU` — git_undo_last_commit
- `gr` — git_revert_file
- `gc` — git_commit
- `gp` — git_push
- `gg` — git_commit_and_push

## Настройки
Источник: `lua/plugins/neo-tree.lua`
- Окно слева, ширина 32.
- Следовать за текущим файлом: включено.
- Скрытые элементы: отображаются.
